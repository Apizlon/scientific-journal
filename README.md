# scientific-journal

## Запуск (Docker)
- Требуется: Docker + Docker Compose.
- Данные: лежат на хосте в `./data/` (папка в `.gitignore`) и переживают перезапуск контейнера.

### Старт
```bash
docker compose up --build
```

Откройте:
- `http://localhost:8080/` (вход)

### Тестовые пользователи
- Администратор: `admin / admin`
- Редколлегия: `redact / redact`

### Регистрация
Для регистрации нужно заполнить все поля: Имя, Фамилию, Почту и Пароль.  
Создаётся пользователь с ролью клиента.

### Структура
- `static/` — статичные страницы (`*.html`)
- `assets/` — стили/скрипты/картинки
- `perl/cgi/` — серверные CGI-скрипты (`*.pl`, доступны как `/cgi-bin/...`)
- `perl/lib/` — общие Perl-модули
- `scripts/` — утилиты (инициализация БД и т.п.)
- `data/` — файлы базы (монтируется в контейнер как `/data`)
